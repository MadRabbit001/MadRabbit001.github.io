אתחול IIS

אתחול לשרת זה נובע מתקלת סנכרון רשתית – בעקבות:​

  1. מס' גרסה באדום​
  2. אין אפשרות לשלוף מידע מהשרת​
  3. תקשורת עם השרת נכשלה ​

כיצד לאתחל אותו? ​

  1. וידוא כתובת IP של שרת IIS POS - יש להשוות כתובת IP בפריורטי מול כתובת בקובץ קונפיגורציה ע"י חיפוש WCFV​
  2. בתוך השרת כדי לאתחל אותו – יש לחפש IIS או להיכנס דרך Task Manager​
  3. תוכנת IIS – Internet Information Services​
  4. רכיבי Application Tools – לוודא שתהליכים בסטטוס Started ​
